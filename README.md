Comparison-of-Region-Based-on-Sales
Overview

This dashboard is designed to help organizations compare sales performance between two selected regions. It provides an interactive and visual way to understand regional sales trends, customer behavior, and overall business performance. By allowing management to select a Primary Region and Secondary Region, the dashboard facilitates a direct comparison between any two regions of interest.

Features

Dynamic Region Selection: Choose any two regions to compare using parameters.

Key Performance Indicators (KPIs): Displays Total Sales, Average Sales per Order, Number of Customers, Number of Orders, Number of Products Sold, and First Order Date for each region.

Visual Comparisons:

Bar Chart: Compares total sales between the two regions.

Line Chart: Shows sales trends over time for both regions to identify patterns, growth, or seasonal fluctuations.

Interactive Dashboard: KPIs and charts update automatically when regions are changed.

Dataset

The dashboard uses the Sample – Superstore dataset, which contains sales transactions, customer details, products, and regional information.

Purpose

The main purpose of this dashboard is to provide management with clear insights into regional performance. By comparing sales and related metrics, stakeholders can:

Identify high-performing regions

Detect underperforming regions

Understand customer engagement and order trends

Make data-driven decisions to improve sales strategy
